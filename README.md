# AWBD-Clinic-Management

## Cerințe Funcționale

- Aplicația permite crearea unui cont de utilizator (register)

- Aplicația permite autentificarea utilizatorilor și accesul pe baza unui token JWT

- Aplicația permite activarea contului printr-un link trimis pe email

- Aplicația permite diferențierea utilizatorilor în funcție de rol (pacient / doctor)

- Aplicația permite pacienților să creeze programări la medici

- Aplicația permite vizualizarea programărilor pentru utilizatorul logat

- Aplicația permite crearea și actualizarea fișei medicale pentru pacienți

- Aplicația permite adăugarea și gestionarea medicamentelor

- Aplicația permite asocierea medicamentelor cu fișa unui pacient

- Aplicația permite pacienților să lase review-uri pentru medici
